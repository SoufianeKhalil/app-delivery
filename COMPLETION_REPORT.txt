ARCHITECTURE
============

STATE MANAGEMENT
----------------
• Context API (CartContext, AuthContext)
• No Redux needed (simpler is better)
• localStorage for cart persistence
• JWT tokens in localStorage

API INTEGRATION
---------------
• Axios for HTTP requests
• React Query for data fetching & caching
• Interceptors for auth headers
• Auto-redirect to login on 401

ROUTING
-------
Public Routes:
  /products          - Product listing (PUBLIC)
  /products/:id      - Product detail (PUBLIC)
  /cart              - Shopping cart (PUBLIC)
  /login             - Login page (PUBLIC)

Protected Routes:
  /checkout          - Checkout form (AUTH REQUIRED)
  /orders            - Order history (AUTH REQUIRED)
  /orders/:id        - Order detail (AUTH REQUIRED)

UI/UX
-----
• Bootstrap 5 for styling
• Responsive grid layouts
• Toast notifications (React Hot Toast)
• Loading spinners
• Error messages
• Form validation feedback
• Visual timeline for order tracking
